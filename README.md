# Usage
<!-- usage -->
```sh-session
$ npm install -g lisef
$ lisef COMMAND
running command...
$ lisef (-v|--version|version)
lisef/0.0.0 win32-x64 node-v8.9.4
$ lisef --help [COMMAND]
USAGE
  $ lisef COMMAND
...
```
